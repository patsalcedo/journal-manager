### Installing Dependecies
To install packages, go to journal-manager then

`npm build`

Then
`npm install`

Afterwards, access the client folder then
`npm install`

Then access backend folder then
`npm install`

### Running the App
To run the app, go to journal-manager then
`npm run dev`
